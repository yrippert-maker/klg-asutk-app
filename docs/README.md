# Документация проекта ФЗ «КЛГ» АСУ ТК

## Техническое задание

**Файл ТЗ:** `ТЗ_КЛГ.pdf` (должен быть размещен в этой папке)

**Информация о ТЗ:**
- Наименование: Техническое задание на создание функциональной задачи «Контроль лётной годности воздушных судов» информационно-аналитической системы регулирования на транспорте (АСУ ТК)
- Заказчик: АО «REFLY»
- Условное обозначение: ФЗ «КЛГ» АСУ ТК

## Соответствие ТЗ

Подробная информация о соответствии проекта техническому заданию находится в файле [ТЗ_соответствие.md](ТЗ_соответствие.md).

## Основные требования ТЗ

### Платформенные решения АСУ ТК
1. **ЦХД АСУ ТК** - Центральное хранилище данных
2. **П-ИВ АСУ ТК** - Протокол интеграции и взаимодействия
3. **АСУ ТК-ИБ** - Информационная безопасность
4. **Информационный портал** - Единая точка входа

### Функциональные требования
- Управление организациями (операторы, MRO, органы власти)
- Управление воздушными судами
- Процесс сертификации организаций по ТО
- Отслеживание технического обслуживания
- Отчеты по дефектам и повреждениям
- И другие функции согласно ТЗ

## Размещение файла ТЗ

Для размещения файла ТЗ выполните:

```bash
cp "/путь/к/файлу/ТЗ КЛГ — копия.pdf" docs/ТЗ_КЛГ.pdf
```

Или разместите файл вручную в папку `docs/`.
